# Multi-Crypto Paper Trading Bot

A **paper trading bot** for DOGE, XRP, and BTC with Kraken price feeds.  
Dashboard shows **equity, USD balance, trades, wins/losses** and has **Deposit/Sell buttons**.  

### Features
- Paper trading (no real money by default)
- Kraken public price feeds
- RSI + EMA trend strategy
- Profit target + stop loss
- Deposit USD manually
- Sell all holdings instantly

### Requirements
- Python 3.10+
- pip install -r requirements.txt

### Run
```bash
python app.py
